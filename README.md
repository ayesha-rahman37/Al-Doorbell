## 📌 Overview
An AI-powered smart doorbell system that uses real-time face detection 
to identify visitors. When someone rings the doorbell, the system 
captures their face and plays an audio alert.

## ⚙️ Features
- Real-time face detection using OpenCV
- Audio doorbell alert (mp3)
- Lightweight and runs on low-end hardware

## 🛠️ Tech Stack
- Python
- OpenCV (face detection)
- Pygame / playsound (audio)

## 🚀 How to Run
1. Clone the repository
   git clone https://github.com/ayesha-rahman37/AI-Doorbell.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the project
   python doorbell.py
